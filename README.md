# Big Brain BackEnd
Facial Recognition App

1. Clone this repo
2. Run `npm install`
3. Run `npm start`

** Make sure you use postgreSQL instead of mySQL for this code base.

 Install Packages 
1. Npm install nodemon
2. Npm install bcrypt
3. Npm install body-parser
4. Npm install cors
5. Npm install express
6. Npm install knex
7. Npm install pg
8. Npm install clarifai
